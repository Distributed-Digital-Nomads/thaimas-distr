"BomEbo FAQ & user support" group chat rules. Edition 1 from 2024-03-08

Before you ask yor question you must:
- read carefully the latest edition of the "Terms and Use conditions" of BomEbo app;
- read carefully the latest edition of the "BomEbo FAQ & user support" group chat rules;
- try to find answer in previous messages reading carefully previous questions and its replays;

You must do NOT:
- include any advertisements in you messages;
- use impolite and rude words and discussion style;
- send message or image that not about BomEbo app use including discuss persons, companies, political subjects etc. It means you can send messages about BomEbo use ONLY;

In case of a user brake the chat rules:
- the user's messages will be locked;
- the user's account in BomEbo may be locked if user's messages brake "Terms and Use conditions" of BomEbo app;

Become official user support
Users that give correct answers to other users can be offered to be an official user support member. The offer will be sent as private message.

These rules can be changed by the chat owners any time.
