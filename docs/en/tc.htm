BomEbo is a free mobile software application that allows Users to order and receive professional massage services at the address specified by the customer.
BomEbo operates within the "Internet of Labour" ecosystem/platform (hereinafter referred to as iol), thus making it an iol-app.
BomEbo does not provide any services itself but serves as a software tool that can be used by both Users and iol-businesses.

iol (The Internet of Labour) is a decentralised/distributed ecosystem/platform designed as one of the solutions to the global unemployment problem. iol is a socio-economic ecosystem/platform.

iol-app refers to software created by an iol-provider using iolBuilder CLI or GUI and operates within the iol according to iol rules and utilising iol capabilities.

iol-provider is a team of one or more authors who have created a functional iol-app. In the case of BomEbo, the iol-provider is DDN vpnf.

iolBuilder is software from iol that allows generating and configuring iol-apps.
iolBuilder CLI is a non-public or internal version that works from the console.
iolBuilder GUI is a public application with a graphical interface that offers convenient and extensive capabilities for generating, configuring, and managing iol-apps as an entire SaaS business.

iol-business is an officially registered organisation representing BomEbo in a specific region in accordance with regional legislation and responsibility.
In the case of BomEbo, this is BomEbo business [region].
iol-businesses can have income from BomEbo commissions/fees and agree on their amount with the iol-provider.

User of BomEbo refers to an individual who launches the BomEbo application and/or interacts with the functionalities of BomEbo.
Login and User identification are carried out through the NiXeL application, which provides access to the User's account in iol.

More information about definitions, terms, and features of iol operations is available in the NiXeL app User Agreement.
